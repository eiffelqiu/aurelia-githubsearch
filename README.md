 
### Quick start
**Make sure you have Node version >= 5.0 and NPM >= 3**

```bash

# install aurelia-cli
npm install aurelia-cli -g
```

```
# install dependencies
npm install
```

```
# start the server
au run --watch

```
go to [http://0.0.0.0:9000](http://0.0.0.0:9000) or [http://localhost:9000](http://localhost:9000) in your browser

```bash
# production
au run --env prod
```
